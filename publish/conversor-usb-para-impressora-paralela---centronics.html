# Descrição do produto

Conecta facilmente computadores de porta USB com periféricos de porta paralela IEEE 1284

Atende aos padrões USB

Conversor de Impressora Paralelo é totalmente compatível com porta USB 1.1 e IEEE 1284 para que as especificações dispositivos operem no desempenho máximo. Auto-alimentado e com baixo consumo de energia, ele permite uma utilização conveniente e confiável em qualquer lugar sem a necessidade de fontes externas de energia.

A instalação rápida e fácil

Plug e Play e compatível com, Windows o Conversor de Impressora Paralelo funciona em qualquer computador equipado com dispositivo USB.

Converte sinal USB para uso com dispositivos IEEE 1284 – suporta transferência de dados de até 12 Mbps
